# Contix - Plataforma de Gestão de Consentimentos LGPD

Plataforma completa para criação e gerenciamento de formulários de consentimento em conformidade com a LGPD.

## Funcionalidades

- 📝 Criação de formulários personalizáveis
- 🔗 Links públicos para compartilhamento
- 👥 Visualização de titulares e consentimentos
- 📊 Indicadores de desempenho
- 📥 Exportação de dados
- 🗑️ Revogação de consentimentos

## Tecnologias

- Next.js 14
- React
- TypeScript
- Tailwind CSS
- Banco de dados em memória (JSON)

## Instalação

```bash
npm install
npm run dev
```

Acesse: http://localhost:3000
